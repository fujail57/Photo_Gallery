# рҹ“ё React Photo Gallery

A responsive **Photo Gallery Web App** built with **React, Vite, and Tailwind CSS**.  
The app fetches photos from a public API, displays them in a responsive grid, allows real-time search by author name, and lets users mark photos as favourites with persistence using `localStorage`.

---

## рҹҡҖ Features

- Fetch **30 photos** from the Picsum API
- Loading and error states
- **Responsive grid layout**
  - Desktop: 4 columns
  - Tablet: 2 columns
  - Mobile: 1 column
- **Real-time search** by author name
- **Favourite photos** using `useReducer`
- **Persist favourites** with `localStorage`
- Custom hook `useFetchPhotos`
- Performance optimization using `useMemo` and `useCallback`
